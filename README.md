# 헥토헬스케어 이벤트 추첨 v4

## 파일 구성
- index.html: 관리자 페이지
- result.html: 참여자 결과 확인 페이지
- announce.html: 게시판 공유용 당첨자 발표 페이지
- style.css
- admin.js
- result.js
- announce.js

## 주요 기능
- 결과페이지 참여자 이름 목록 표시
- 이름 입력 후 당첨/미당첨 애니메이션
- 관리자 페이지에서 당첨자 발표 링크 생성
- 게시판 첨부용 PNG 이미지 저장
- 게시판 첨부용 WebM 영상 저장
